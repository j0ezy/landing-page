# landing-page

Created a landing page from a picture

First created teh HTML boilerplate and created most of the divs, header, footer. Next I added the content, so mostly just text in this case.

Then I created my style.css file and started of with just the positioning of elements. Then after i was okay with the layout I added all the font sizes, font family, font weight, colors and borders.

After completing all that i tweaked some paddings, position of elements. 

Think to remember is to use * at start to remove all default paddings and margins, otherwise you'll get some issues.
